# Quick Start

```bash
npm install       # install dependencies
npm start         # dev server at http://localhost:4200
npm run build     # production build → dist/
```

## Notes
- Active branch: **main** — push directly, no PRs unless asked.
- Local dev runs in the browser (mock adapter active by default).
- Web Bluetooth (`WebBluetoothElm327Adapter`) requires **Chrome or Edge** and either `localhost` or an **HTTPS** origin — it will not work on plain HTTP.
